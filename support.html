Questions fréquentes

Où sont stockées mes données ?

Toutes vos données (entraînements, nutrition, objectifs) sont stockées sur votre compte iCloud personnel via Apple CloudKit. Personne d'autre — y compris les développeurs de Forge — n'y a accès.

Forge collecte-t-il mes données personnelles ?

Non. Forge ne collecte, ne transmet et ne vend aucune donnée personnelle à des tiers. Vos données restent sur votre iCloud.

Comment synchroniser entre mes appareils Apple ?

La synchronisation est automatique via iCloud. Assurez-vous d'être connecté avec le même Apple ID sur tous vos appareils et qu'iCloud Drive est activé dans les réglages.

Comment supprimer mes données ?

Vous pouvez supprimer vos données directement depuis l'application, ou en supprimant Forge et en effaçant les données iCloud associées depuis les réglages iCloud de votre iPhone.

L'app est-elle gratuite ?

Forge est disponible gratuitement avec toutes les fonctionnalités essentielles. Des fonctionnalités premium pourront être proposées à l'avenir via achat intégré.

La fonction IA nécessite-t-elle une clé API ?

Les fonctionnalités IA (reconnaissance photo, lecture d'étiquettes, suggestions) nécessitent une clé API OpenAI personnelle. Vous pouvez la configurer dans les réglages de l'app. Sans clé, toutes les autres fonctionnalités restent disponibles.
